The good ol’ apply functions

I first met this class in my true isolation days. I was on scholarship at a biomedical institute where it felt like nobody thought a second-year university student should be thus polluting their shades.

I want to know how the function and its arguments can be vecorised.

Should I be experimenting with a list or with a tibble? Let’s try a tibble first, hopefully I’m right in thinking of it as a special type of list. One where the cardinality of every element is the same.

# testing tibble
(tt <- tibble(x = c(3, 4), y = c(2, 5)))
## # A tibble: 2 x 2
##       x     y
##   <dbl> <dbl>
## 1     3     2
## 2     4     5

Apply a function to each column

lapply(tt, function(x) {x + 1})
## $x
## [1] 4 5
## 
## $y
## [1] 3 6

So this seems to work as I understand it. Can I vectorise?

fun_one <- function(x) {x + 1}
fun_two <- function(x) {x^2}

lapply(tt, FUN = c("fun_one", "fun_two"))
